# Roco Box Detector 使用说明

## 快速开始

1. 解压后双击 `RocoBoxDetector.exe`
2. 屏幕变暗，用鼠标拖拽框选战斗时惊喜盒子出现的区域，可以先进入一场战斗再确认框选位置，按 `Enter` 确认
3. 检测自动开始，屏幕出现悬浮窗

## 悬浮窗按钮

| 按钮 | 功能 |
|------|------|
| `⬤` / `○` | 调试截图开关。平常关着，需要看匹配效果时打开 |
| `🔍` / `◌` | 预览窗口开关。显示蓝框/黄框/绿框位置，调试用 |
| `▣` / `□` | 画面覆盖框开关。把检测框直接画在游戏上 |
| `↻` | 重新框选游戏区域 |
| `⚙` | 打开设置面板 |
| `✕` | 退出程序 |

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+R` | 重新框选 ROI |
| `Ctrl+Shift+Q` | 退出程序 |

## 识别历史面板

屏幕右侧的半透明面板，显示历史识别结果。

- 拖动标题栏可以移动位置
- 右下角拖拽可以调整大小
- 每个图案有独立颜色
- 最新结果字号最大
- 底栏分 ROI1 / ROI2 统计各图案出现次数
- `清空` 删除记录，`⚙` 打开设置，`✕` 隐藏面板

## 首次设置（重要）

### 1. 分辨率预设
打开 `⚙` 设置面板 → **Runtime** 标签页 → **游戏分辨率预设**，选择你游戏实际运行的分辨率：

| 预设 | 我的游戏窗口是 |
|------|--------------|
| 720p | 1280×720 左右 |
| 1080p | 1920×1080 左右 |
| 2K | 2560×1440 左右 |
| 4K | 3840×2160 左右 |

**注意**：选游戏窗口的实际分辨率，不是显示器的物理分辨率。

### 2. 检查黄框
蓝框命中后，预览窗口会出现两个黄框。确认黄框是否对准了你要识别的图案。如果偏了，在设置面板的 **黄框1 / 黄框2** 标签页调整位置。

## 设置面板

`⚙` 打开，各标签页功能：

| 标签页 | 改什么 |
|--------|--------|
| Anchor | 蓝框（盒子整体）的匹配参数和模板列表 |
| 黄框1 | 第一个图案区域的位置和大小 |
| 黄框2 | 第二个图案区域的位置和大小 |
| 样本1 | 第一个图案的模板和匹配参数 |
| 样本2 | 第二个图案的模板和匹配参数 |
| Runtime | 帧率、分辨率预设、跳帧间隔等 |
| Debug | 预览窗口、调试截图开关 |
| 悬浮窗 | 顶部状态栏样式 |
| 提示文字 | 识别历史面板的样式和颜色 |

**改完记得点"保存并应用"。**

## 注意事项

- **单人 / 双人抓宠**：单人战斗和双人组队战斗的相机视角不同，盒子的屏幕位置会有偏移。建议分别框选一次 ROI 并保存各自的 config，或者切换后按 `Ctrl+Shift+R` 重新框选。
- **游戏分辨率变更**：调整游戏分辨率（窗口大小）后，画面比例发生变化，**必须重新框选 ROI**，否则蓝框可能识别不到。框选后去 Runtime 确认分辨率预设匹配当前游戏窗口。
- **设置面板改完后务必点"保存并应用"**，否则重启后配置丢失。

## 常见问题

**Q: 蓝框一直识别不到？**
- 检查 Runtime → 分辨率预设是否选对
- 调整 Anchor 标签页的"匹配阈值"（降低到 0.55-0.60）
- 调整"粗筛阈值"（降低或设为 0 关闭）

**Q: 蓝框识别到了但绿框匹配不准？**
- 降低样本1/样本2 的"匹配阈值"
- 调整黄框1/黄框2 的位置使其对准目标图案
- 增加模板样本的数量和多样性

**Q: 游戏全屏时悬浮窗看不见？**
- 游戏设为无边框窗口模式即可

**Q: 预览感觉卡顿？**
- Runtime → Anchor跳帧间隔 调大（如 2 或 3）
- 关掉预览窗口（🔍）
- 关掉画面覆盖框（▣）
- 关掉调试截图（⬤）

**Q: 想要识别更多图案？**
- 样本1 / 样本2 标签页 → `+ 新增图案` → 添加模板图片 → 保存并应用
- 黄框2 默认关闭，需要先在黄框2 标签页勾选"启用区域二"

**Q: 程序打不开？**
- 不要放在中文路径下（虽然已尽量兼容，但仍可能有坑）
- 管理员权限运行试试
