# 洛克王国盒子检测器 (Roco Box Detector)

本小软件由Deepseek+CC+Chatgpt+Gemini联合制作，我负责提供想法，提供数据集和校准参数
相信你们S2赛季抓异色的时候总会走神忘记看盒子上的精灵属性，特别是狼熊梨之类单抓可回本的精灵，因此在基本没有编程基础的背景下，我和AI军团搏斗了5天写出来了这个小软件
注意！！！！！由于个人能力不足以及数据集缺失，本软件仍然存在识别率上的缺陷，整体识别成功率约为75%左右，欢迎各位提交代码以改善使用体验

## 功能亮点

- **双区域独立识别** — 同时识别盒子中下层的两个属性标志（如属性 + 种族），结果以 `普通 + 物防` 格式展示
- **多帧采样投票** — 蓝框锁定后截取多帧子图，投票决定最终结果，2 票即提前终止，抗干扰能力强
- **自适应缩放** — 模板搜索范围根据框选区域自动缩放，不同分辨率下都能准确定位
- **画面覆盖框** — 检测框（蓝/黄/绿）直接覆盖在游戏画面上，方便实时观察匹配效果
- **识别历史面板** — 半透明悬浮面板，显示历史识别记录，每种图案独立颜色，支持拖动和缩放
- **可视设置面板** — 无需手动编辑配置文件，所有参数（阈值、缩放、模板、Sub-ROI 位置等）可视化调整
- **高斯权重蒙版** — Pattern 模板匹配时中心区域权重更高，边框背景的干扰被大幅压低
- **极速截屏** — 采样阶段直接截取子区域，单帧截屏 < 5ms，严格按设定间隔执行

## 工作原理

```
框选 ROI → anchor(蓝框) 匹配 → 命中锁定 → 延迟 0.3s → 连续截取 5 帧子区域
→ 分批匹配 pattern(绿框) → 多帧投票 → 2 票通过即输出结果
```

Anchor 命中后锁定位置，采样期间不重新检测蓝框，截屏和匹配解耦——截屏按精确间隔执行，匹配延迟到采样完成后一次性运行。

## 安装

```bash
pip install -r requirements.txt
```

依赖：`opencv-python` `numpy` `PyQt5` `mss` `keyboard`

## 使用

```bash
python main.py
```

1. 启动后屏幕变暗，鼠标拖拽框选游戏画面区域
2. Enter 确认，Esc 取消
3. 检测自动开始，屏幕出现悬浮窗

| `⬤`/`○` | 调试截图开关（红色=开） |

| `▣`/`□` | 画面覆盖框开关（橙色=开） |

## 模板准备

### Anchor 模板（惊喜盒子整体——建议选取旋转完成后三个图标发光的几帧）
位置：`templates/box_anchor/`

- 包含三层盒子主体、顶部结构、左右边缘
- 不要包含太多草地、天空、粒子特效
- 建议 10~12 张覆盖主要光照/角度变化

### Pattern 1 模板（精灵类型中层图案）
位置：`templates/patterns/`（区域一）

- 只包含中层侧面的目标图标
- 含少量周围边框，图案特征完整
- 不同光照/颜色变化各一张

### Pattern 2 模板（精灵加成下层图案）
位置：`templates/patterns_2/`（区域二）

- 只包含下层侧面的目标图标
- 含少量周围边框，图案特征完整
- 不同光照/颜色变化各一张

## 重要：首次使用

### 1. 设置游戏分辨率预设
打开设置面板（⚙）→ Runtime 标签页 → "游戏分辨率预设"，选择你游戏运行的分辨率。这会自动调整模板缩放参数。

| 预设 | 适用场景 |
|------|---------|
| 720p | 720p 窗口化 |
| 1080p | 1080p 窗口化 |
| 2K（默认） | 1440p / 2K 窗口化 |
| 4K | 4K 窗口化 |

> 如果游戏缩放到了非默认尺寸（比如 2K 屏开 1080p 窗口），选实际游戏窗口分辨率，不要选显示器分辨率。

### 2. 确认黄框位置
首次框选 ROI 后，在预览窗口确认两个黄框是否对准了目标图案。如果偏移，在设置面板的黄框1 / 黄框2 中调整 x/y/w/h 比例。

### 3. cooldown 期间无反应
识别成功后进入 cooldown（冷却期，默认 1.5s），期间不会触发新一轮识别。蓝框消失后重新出现才会开始下一次检测。

## 性能

- 截屏阶段单帧 < 5ms，5 帧总截屏 < 0.1s
- 匹配阶段每帧 ~0.2s，提前投票可跳过后续帧
- Anchor 粗筛 + 跳帧：空帧 ~20ms、有盒子帧 ~300ms
- 关预览窗口可节省少量 CPU，关画面覆盖框不影响性能（仅绘制开销）
- 全屏独占模式下所有悬浮窗均无法显示，建议游戏使用无边框窗口模式

## 调参优先级

1. **选对分辨率预设** → Runtime 标签页，这是最重要的第一步
2. 确认蓝框命中 → 调整 anchor threshold 和模板质量，粗筛阈值太高会漏掉盒子
3. 确认黄框对准目标 → 调整 sub_roi / sub_roi_2 的四个 ratio
4. 提高投票通过率 → 降低 min_average_score、增加样本多样性
5. 提速 → 减少 scale_steps、降低 max_frames、增大 anchor_skip_frames
